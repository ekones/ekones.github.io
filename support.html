How to Play - Mathze Game

Objective: The objective of the game is to select a combination of numbers and arithmetic operators (+ and -) to make the resulting sum equal to zero. As you progress through the levels, the board size increases, adding more complexity to the game.

Game Elements:

  • Board: The game board consists of a grid of numbered tiles and arithmetic operator tiles (+ and -).
  • Difficulty Levels: The game offers four difficulty levels (Easy, Medium, Hard, and Expert), which can be selected from a dropdown menu.
  • Controls: You can restart the game, move back to the previous level, or progress to the next level using the provided controls.

How to Play:

  1. Start the game by selecting a difficulty level from the dropdown menu.
  2. Observe the game board and plan your strategy to make the sum zero.
  3. Tap a number tile to select it. The selected number tile will change color.
  4. Tap an arithmetic operator (+ or -) tile to select it. The selected operator tile will change color.
  5. Tap another number tile to apply the selected operator. The sum will be calculated, and the new number will replace the second selected number tile. The first number tile and the operator tile will be marked as used and become unselectable.
  6. Continue to select number and operator tiles to reach the sum of zero. The game is won when all the remaining number tiles have a value of zero.
  7. If only one tile is left, and the board cannot be further modified, the game is lost.

Tips:

  • Carefully plan your moves to avoid getting stuck with an unsolvable board.
  • Keep an eye on the difficulty level; higher difficulty levels will introduce a larger range of numbers, making the game more challenging.

With this guide, you are now ready to test and improve your arithmetic skills in the Simple Math Game. Good luck, and have fun!
